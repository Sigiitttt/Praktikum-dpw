Tentang Saya

Halo! Nama saya Moch Sigit Aringga, dan saya adalah mahasiswa Teknik Informatika yang sedang menempuh semester 2 di Universitas Trunojoyo Madura. Saya tertarik dengan teknologi. Ketertarikan ini membawa saya untuk masuk prodi Teknik Informatika, di mana saya dapat mengeksplorasi dan mengembangkan kemampuan saya dalam pemrograman, pengembangan perangkat lunak, dan desain sistem. Sebagai mahasiswa Teknik Informatika, saya telah menguasai berbagai bahasa pemrograman seperti Python, R, HTML, dan CSS. Saya juga memiliki pengalaman dalam mengembangkan aplikasi web, serta memahami konsep-konsep dasar jaringan dan data science. Proyek-proyek yang saya kerjakan selama studi saya membantu saya untuk mempraktikkan pengetahuan teoretis dalam konteks dunia nyata. Saya juga terlibat dalam berbagai kegiatan di luar kampus, seperti mengikuti kursus online dan bergabung dengan komunitas pengembang. Kegiatan-kegiatan ini tidak hanya memperluas jaringan saya dengan individu yang memiliki minat yang sama. Di luar akademik, saya memiliki beberapa hobi yang saya tekuni. Saya senang bermain olahraga untuk menjaga kebugaran fisik, membaca buku untuk memperluas wawasan. Terima kasih telah mengunjungi halaman saya! Jangan ragu untuk menghubungi saya.
